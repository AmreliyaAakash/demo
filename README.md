# ☕ BODREN Coffee Artisans - Setup Guide

Welcome! This guide is designed to help you get the **BODREN Coffee Landing Page** up and running on your computer, even if you're not a technical person.

---

## 🚀 Quick Start (In 3 Simple Steps)

### Step 1: Install the "Engine" (Node.js)
To run this website, your computer needs a small program called **Node.js**.
1. Go to [nodejs.org](https://nodejs.org/).
2. Click the button that says **"LTS"** (it stands for "Long Term Support" and is the safest version).
3. Download and install it just like any other app (keep clicking "Next").

### Step 2: Open the Project
1. Open **VS Code** (the program you use to see this code).
2. Go to `File` > `Open Folder...` and select the **Demo1** folder.
3. Look at the top menu of VS Code and click **Terminal** > **New Terminal**. A small black box will open at the bottom.

### Step 3: Start the Website
In that black box (terminal), type these two commands one by one:

1. Type this and press **Enter**:
   ```bash
   npm install
   ```
   *(Wait about 1-2 minutes while it downloads the necessary parts. You'll see moving bars.)*

2. Type this and press **Enter**:
   ```bash
   npm run dev
   ```
   *(You will see a link like `http://localhost:5173/`. **Click that link** or copy it into your browser!)*

## 🎨 What is this?
This is a premium, cinematic 3D landing page for a coffee shop. 
- **The 3D Cup**: It moves, rotates, and shrinks as you scroll.
- **Cinematic Reveals**: Text and images appear gracefully.
- **The Blur Finish**: As you reach the contact details at the bottom, the 3D scene blurs out to help you focus on the information.

## 🛠️ Need Help?
If the website doesn't show up:
- Make sure you followed **Step 1** correctly.
- If you see an error in the black box, try closing VS Code and opening it again.
